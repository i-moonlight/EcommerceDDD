Welcome to Ecommerce DDD!

This single-page application was built using:

  • ASP.NET Core and C# for cross-platform server-side code with:
    • .NET Core 3.1
    • Entity Framework Core 3.1
    • ASP.NET WebApi Core with JWT Bearer Authentication
    • ASP.NET Identity Core
    • MediatR
    • Automapper
    • Fluent Validation

  • Angular and TypeScript for client-side code with:
    • Bootstrap for layout and styling
    • NgBootstrap
    • Font Awesome
    • Toastr
    • Angular JWT
  • Check the API using Swagger!
  • Check the HeathChecks!
