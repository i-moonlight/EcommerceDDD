    • ASP.NET Core Identity
    • SignalR Core
    • JWT Bearer Authentication
    • BuildingBlocks.CQRS 2.x
    • MediatR
    • Fluent Validation
    • Automapper
    • NSubstitute

    • HealthChecks
    • Docker Compose
  • Angular 12 and TypeScript for the front-end with:
    • NgBootstrap
    • Font Awesome
    • Toastr
    • Angular JWT
